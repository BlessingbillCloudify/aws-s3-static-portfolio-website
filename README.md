# AWS S3 Static Portfolio Website

## 📌 Project Overview

This project is a personal portfolio website built using HTML, CSS, and JavaScript and deployed using Amazon S3 Static Website Hosting.

It demonstrates how to host a static website on AWS without using a web server.

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
- Amazon S3
- Git
- GitHub

---

## ✨ Features

- Responsive portfolio design
- Professional profile image
- About Me section
- Technical Skills section
- Projects section
- Hosted on Amazon S3

---

## 📚 What I Learned

During this project I learned how to:

- Create an Amazon S3 bucket
- Configure Static Website Hosting
- Upload website files to S3
- Configure Bucket Policies
- Make an S3 website publicly accessible
- Troubleshoot HTTP vs HTTPS issues
- Deploy a live website using AWS
